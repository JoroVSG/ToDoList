Welcome in ToDo Api powered by Giraffe (F#)

(swagger/openapi integration is on it's way)


It uses "Basic" schema for authentication, please, contact the correspondant to a the token

The responses are formatted in JSON API standard

ToDo api has the following endpoints

  • GET /todo - returns list of todos
  • GET /todo/{idTodo} returns single todo with given id
  • POST /todo creates todo taken from the body of the request
  • PUT /todo update(mutate) todo taken from the body of the request
  • DELETE /todo/1 deletes todo with given id

Sample request:

curl --location --request GET '/todo/1' --header 'Authorization: Basic YzQ2ZWVhM2QtY2MxNi00MG....'

"jsonapi": { "version": "1.0.0", "meta": null }, "data": { "id": 1, "name": "New Task", "description": "This task was added by API call", "dueDate": "2020-09-19T02:05:49.933", "isDone": true }, "included": null
